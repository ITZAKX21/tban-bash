# TBAN-BASH USER
TERMUX TERMINAL EDIT BASH USER


TERMUX SETUP COMMANDS

````
pkg up -y
pkg i python -y
pip install lolcat
````

BASH FILE SETUP

````
cd ..
cd usr
cd etc
````

NOW MAKE YOUR LOGO FILE

````
nano logo.py
````

#NOW PASTE YOUR FULL LOGO HERE & PRESS CTRL THEN Z THEN ENTER
#After create logo file type

````
pwd
````

COPY YOUR DIRECTOTY & OPEN BASH.BASHRC FILE BY USING THIS COMMAND

````
nano bash.bashrc 
````

THEN TYPE

````
python your file directory/filename
````

Example

````
python /data/data/com.termux/files/usr/etc/logo.py
````

NOW REOPEN YOUR TERMUX
